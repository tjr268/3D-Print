# Summary

Grafted image from official Audi emblem / logo published in 2009

Assembly shown as two colors as if two different color filaments used. Intended to be printed as assembly using two filament 3D printer due to number of free bodies in the audi ring frame. Slip fit between two parts. 

high resolution, true aspect ratio, two pieces.

Able to be scaled up, resolution of part based on printer.
